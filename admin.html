All products

To publish: your edits save in this browser automatically. Either hit 🚀 Publish (after connecting a repo under ⚙ GitHub) to push straight to GitHub, or hit Export products.js, drop the file into assets/js/, and push it yourself. Either way, that's what your live visitors see — localStorage here is just your private workspace.

GitHub publishing

Publish writes products.js straight to your repo via the GitHub API — no manual export/copy/push. Your token is stored only in this browser's local storage, so use a fine-grained personal access token scoped to just this repo's contents (not a broad admin token), and skip this on a shared computer.
